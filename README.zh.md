LOICollectionA-Expand 是基于 LOICollectionA 的扩展插件，主要目的是添加部分受制于原项目架构而无法实现的功能。  

目前项目仍然处于开发阶段，功能尚未完善。

## 实现功能模块

> 以下功能模块均可在配置文件中进行配置开启或关闭。

- [x] GMLIBExpand
- [x] ConfigExpand

## 安装插件

1. 在服务器目录中执行命令

    ```cmd
    lip install github.com/loitietu/LOICollectionA-Expand
    ```

2. 启动服务器。
3. 等待输出加载文本。
4. 完成安装。

## 本地编译

打开本地命令提示符(`cmd`)并执行以下命令：

```cmd
git clone https://github.com/loitietu/LOICollectionA-Expand.git
cd LOICollectionA-Expand
xmake repo -u
xmake
```
